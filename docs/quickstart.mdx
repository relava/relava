# Quick Start

## 1. Install Relava

Install the Relava CLI using the install script:

<Tabs>
  <Tab title="macOS / Linux">
    ```bash
    curl -fsSL https://raw.githubusercontent.com/relava/relava/main/scripts/install.sh | sh
    ```
  </Tab>
  <Tab title="Windows">
    ```powershell
    irm https://raw.githubusercontent.com/relava/relava/main/scripts/install.ps1 | iex
    ```
  </Tab>
  <Tab title="Cargo">
    ```bash
    cargo install relava
    ```
  </Tab>
</Tabs>

See the [Installation](/installation) page for more options and details.

## 2. Start the Registry Server

Relava uses a local registry server to store and serve resources. Start it as a background daemon:

```bash
relava server start --daemon
```

The server runs at `http://localhost:7420` by default. You can verify it's running:

```bash
relava server status
```

<Tip>
  The server also hosts a web GUI at `http://localhost:7420` for browsing and searching resources visually.
</Tip>

## 3. Initialize Your Project

Navigate to your project directory and initialize Relava:

```bash
cd my-project
relava init
```

This creates a `relava.toml` manifest file in your project root:

```toml
agent_type = "claude"

[skills]

[agents]

[commands]

[rules]
```

## 4. Install a Resource

Install a skill from the registry:

```bash
relava install skill denden --save
```

The `--save` flag writes the resource and its pinned version to `relava.toml`:

```toml
agent_type = "claude"

[skills]
denden = "1.2.0"
```

The skill files are installed to `.claude/skills/denden/`.

## 5. Install More Resources

```bash
# Install an agent
relava install agent debugger --save

# Install a command
relava install command commit --save

# Install a rule
relava install rule no-console-log --save
```

## 6. Share Your Setup

Commit `relava.toml` to version control. Collaborators reproduce your exact setup with:

```bash
relava install relava.toml
```

This installs every resource declared in the manifest at the pinned versions.

## 7. Explore the Registry

```bash
# Search for resources
relava search notify

# List installed resources
relava list

# Get details about a resource
relava info skill denden
```
